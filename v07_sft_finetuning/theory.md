# SFT 监督微调原理

## 1. SFT (Supervised Fine-Tuning)

在预训练模型基础上，用高质量的指令-回答数据进行微调，让模型学会按指令完成任务。

## 2. LoRA 低秩适配

$$W = W_0 + \Delta W = W_0 + BA$$

其中 $B \in \mathbb{R}^{d \times r}$, $A \in \mathbb{R}^{r \times d}$, $r \ll d$。

只训练 A 和 B（秩为 r 的低秩矩阵），冻结原始权重 $W_0$。

**参数量对比**: 全参微调 $d^2$ vs LoRA $2dr$。当 $r=16, d=4096$ 时，仅需 0.78% 参数。

## 3. QLoRA

LoRA + 4-bit 量化：
1. 将预训练权重量化为 4-bit (NF4)
2. 计算时反量化回 FP16
3. 只更新 LoRA 的 FP16 参数

显存节省：7B 模型从 ~28GB 降到 ~6GB。

## 4. 广告场景 SFT 任务

- 广告创意理解：描述广告图片内容
- 商品属性提取：从广告中提取品牌、产品、卖点
- 广告文案生成：根据图片生成广告文案
- 相关性判断：判断广告与搜索 query 的相关程度
