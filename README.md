# 🚀 多模态大模型从 0 到 1 学习工程

> 从 Transformer 基础到广告商业场景端到端多模态 Pipeline 的完整学习路径 — **25 个原理教学 + 10 个实践操练版本，400+ 个文件**

## 项目简介

本项目是一套**渐进式多模态大模型学习工程**，包含两大系列：

### 📖 原理教学系列 (v01-v25)
面向理解原理的工程师/研究员，通过 **25 个递进版本**，从 Transformer 基础到端到端广告 Pipeline，用迷你模型 + 合成数据讲解每个核心概念。

### 🔥 实践操练系列 (p01-p10) — NEW!
面向实际操作的工程师，在**单卡 GPU (4090/A6000)** 上手把手跑通大模型全套流程：继续预训练 → SFT → DPO → RL (GRPO) → MLLM → 量化 → 部署 → 评测。使用真实模型 (Qwen2.5-0.5B) + 真实数据 + 真实 GPU，覆盖多种方法对比和参数消融，做到**练完即该领域专家**。

**每个版本都配备交互式 HTML 可视化动画** — 📖 [打开导览首页 index.html](./index.html)
